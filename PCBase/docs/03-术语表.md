# 03 术语表

本章不是百科全书，而是为 PC-based 运动控制初学者准备的“高频词速查表”。  
你不需要一次记住所有术语，但至少要知道它们分别属于哪一层：机械层、控制层、总线层、驱动层、安全层。

## 1. 机械与轴概念

| 术语 | 含义 | 为什么重要 |
| --- | --- | --- |
| Axis / 轴 | 一个可独立控制的位置自由度，例如 X、Y、Z、R | 运动控制的最小单元 |
| Stroke / 行程 | 机构可以移动的有效范围 | 决定软限位和回零范围 |
| Coordinate System / 坐标系 | 位置表达方式，如机床坐标、工件坐标 | 多轴联动和视觉标定离不开它 |
| Resolution / 分辨率 | 系统能区分的最小位移单位 | 影响定位细腻程度 |
| Accuracy / 精度 | 实际位置接近目标位置的程度 | 不等于重复定位精度 |
| Repeatability / 重复定位精度 | 多次到同一点时的一致性 | 自动化设备里通常比绝对精度更关键 |
| Backlash / 背隙 | 丝杆、齿轮等机械传动反向切换时的空程 | 会直接影响反向定位和插补质量 |
| Load / 负载 | 电机和机构需要驱动的质量与惯量 | 决定选电机、驱动器和加速度 |
| Lead / 导程 | 丝杆转一圈移动的距离 | 单位换算的核心参数 |

## 2. 基本运动方式

| 术语 | 含义 | 为什么重要 |
| --- | --- | --- |
| Jog / 点动 | 按住就动，松开就停 | 联调和维护最常用 |
| Inch / 寸动 | 每次按键只走固定短距离 | 微调工位常用 |
| Absolute Move / 绝对定位 | 移动到绝对坐标值 | 配方定位和工位切换常用 |
| Relative Move / 相对定位 | 相对当前位置再移动一段距离 | 补偿和步进式动作常用 |
| Velocity Move / 连续速度运动 | 给定速度一直走，直到停止 | 输送、搜索、调试常见 |
| Homing / 回零 | 建立参考原点的过程 | 没有回零，坐标体系通常不可靠 |
| Interpolation / 插补 | 多轴按规划轨迹同步运动 | 直线、圆弧、轨迹跟随的基础 |
| Contouring / 轮廓加工 | 多轴连续轨迹运动 | 点胶、焊接、加工等场景常见 |
| Blending / 圆滑过渡 | 段与段之间不断速或少降速连接 | 提升节拍和平滑性 |
| Look-ahead / 前瞻 | 提前分析后续轨迹决定速度变化 | 高速轨迹跟随时很重要 |

## 3. 速度规划与控制参数

| 术语 | 含义 | 为什么重要 |
| --- | --- | --- |
| Velocity / 速度 | 单位时间内的位置变化量 | 基础参数 |
| Acceleration / 加速度 | 速度变化率 | 影响启动冲击和节拍 |
| Deceleration / 减速度 | 速度下降率 | 影响停车距离和稳定性 |
| Jerk / 加加速度 | 加速度变化率 | 决定冲击感和平滑性 |
| T-Curve / 梯形速度曲线 | 常见的加减速方式 | 实现简单 |
| S-Curve / S 曲线 | 更平滑的速度曲线 | 高速高精设备更常用 |
| Settling Time / 稳定时间 | 到位后进入稳定误差带所需时间 | 节拍分析要看它，不只看“到位” |
| Following Error / 跟随误差 | 指令位置与实际位置的差值 | 闭环系统重要报警依据 |
| Feed Forward / 前馈 | 根据模型提前补偿控制输出 | 提高动态响应 |
| PID | 比例、积分、微分控制 | 几乎所有闭环系统都会遇到 |

## 4. 脉冲控制相关术语

| 术语 | 含义 | 为什么重要 |
| --- | --- | --- |
| Pulse/Dir | 脉冲 + 方向控制方式 | 最常见的步进/脉冲伺服控制方式 |
| CW/CCW | 正反向各一路脉冲 | 某些驱动器或老系统常见 |
| PPU / Pulses Per Unit | 每个工程单位对应的脉冲数 | 毫米、角度换算的关键 |
| E-Gear / 电子齿轮 | 用电子方式定义输入脉冲与内部位置比例 | 适配驱动器和机构换算 |
| Compare Output / 比较输出 | 到达指定位置时输出硬件触发信号 | 飞拍、打标、喷胶常用 |
| Latch / 锁存 | 在外部信号到来时捕获当前位置 | 测长、对位、基准检测常见 |
| MPG / 手轮 | 手动脉冲发生器 | 调机时非常常见 |

## 5. 编码器与反馈相关术语

| 术语 | 含义 | 为什么重要 |
| --- | --- | --- |
| Encoder / 编码器 | 检测位置和速度的反馈器件 | 闭环控制核心之一 |
| Incremental Encoder / 增量式编码器 | 上电后没有绝对位置，需要回零 | 工业现场很常见 |
| Absolute Encoder / 绝对值编码器 | 上电即可知道绝对位置 | 高端伺服和多圈系统常见 |
| A/B/Z 相 | 增量编码器常见三路信号 | A/B 用于计数，Z 用于基准索引 |
| Linear Scale / 光栅尺 | 直线位置高精度反馈元件 | 高精平台常用 |
| Semi-closed Loop / 半闭环 | 反馈取自电机端编码器 | 成本低、结构简单 |
| Full-closed Loop / 全闭环 | 反馈取自工作台或光栅尺 | 精度高，但调试更复杂 |

## 6. I/O、联锁与安全术语

| 术语 | 含义 | 为什么重要 |
| --- | --- | --- |
| ORG / 原点信号 | 回零参考开关 | 决定回零过程 |
| +EL / -EL | 正限位 / 负限位 | 防止越界 |
| EMG / 急停 | 紧急停止链路输入 | 不是普通软件停止 |
| Servo ON / SVON | 驱动器使能 | 没有使能，轴不会真正响应 |
| ALM / Alarm | 驱动器或轴报警 | 现场最常见故障入口 |
| INP / In Position | 到位信号 | 工艺动作常以此做联锁 |
| STO / Safe Torque Off | 安全转矩关断 | 通过安全链切断电机扭矩输出 |
| Interlock / 联锁 | 某条件不满足时禁止动作 | 保证设备和人员安全 |
| Watchdog / 看门狗 | 检测控制器或软件是否异常卡死 | 提高失控防护能力 |

## 7. EtherCAT 与总线术语

| 术语 | 含义 | 为什么重要 |
| --- | --- | --- |
| EtherCAT | 面向实时工业控制的高速以太网总线 | 现代 PC-based 运动控制主流方案 |
| Master / 主站 | 发起总线周期通信的控制端 | 通常是 PC 侧主站卡或软主站 |
| Slave / 从站 | 驱动器、I/O、编码器等现场设备 | 实际执行控制 |
| Distributed Clocks / 分布式时钟 | 让所有从站共享高精度同步时间基准 | 多轴同步核心机制 |
| Cycle Time / 周期时间 | 主站刷新总线数据的时间间隔 | 影响控制实时性 |
| Jitter / 抖动 | 周期时间的不稳定性 | 抖动大时同步和轨迹质量都会受影响 |
| CoE / CANopen over EtherCAT | 在 EtherCAT 上传输 CANopen 对象字典和服务 | 许多驱动器参数访问都靠它 |
| PDO / Process Data Object | 周期性交换的实时数据 | 常用来传控制字、状态字、实际位置 |
| SDO / Service Data Object | 非周期参数读写服务 | 用于配置参数和诊断 |
| Object Dictionary / 对象字典 | 设备参数地址空间 | 调 CiA 402 设备时必须会看 |
| Sync0 / Sync1 | EtherCAT 从站同步中断信号 | 常用于驱动器采样对齐 |

## 8. CiA 402 驱动控制相关术语

很多 EtherCAT 伺服驱动器会遵循 `CiA 402` 设备模型。初学者常见缩写如下：

| 术语 | 含义 | 为什么重要 |
| --- | --- | --- |
| CiA 402 | 伺服驱动器通用设备轮廓 | 不同品牌驱动器可以按类似方式接入 |
| Controlword | 控制字 | 用于使能、停止、复位、模式控制 |
| Statusword | 状态字 | 用于读取驱动器当前状态 |
| Operation Mode | 运行模式 | 决定驱动器当前接受什么类型指令 |
| CSP | Cyclic Synchronous Position | 周期同步位置模式，最常见 |
| CSV | Cyclic Synchronous Velocity | 周期同步速度模式 |
| CST | Cyclic Synchronous Torque | 周期同步转矩模式 |
| PP | Profile Position | 轮廓位置模式 |
| PV | Profile Velocity | 轮廓速度模式 |
| HM | Homing Mode | 回零模式 |

如果你在 EtherCAT 项目里反复遇到“能上 OP 但不动”，往往问题就在：

- 模式没切对
- 控制字顺序不对
- PDO 没映射完整
- 目标位置虽然下发了，但使能链没打通

| 术语 | 含义 | 为什么重要 |
| --- | --- | --- |
| Gantry / 龙门 | 两轴机械绑定成一个逻辑轴 | 双驱龙门平台必备概念 |
| Electronic Gearing / 电子齿轮 | 从轴按比例跟随主轴 | 送料、跟随、同步常用 |
| Electronic Cam / 电子凸轮 | 从轴按主轴位置曲线跟随 | 包装、印刷、贴装等设备常见 |

## 10. 初学者最容易混淆的几组词

### 精度 vs 重复定位精度

- 精度：你命令到 `100.000 mm`，实际到了多少
- 重复定位精度：你反复跑到同一点，每次是否一致

自动化设备里，重复定位精度常常更能决定工艺稳定性。

### 回零 vs 清零

- 回零：机械轴去找真实参考点
- 清零：软件把当前位置改成某个坐标值

两者完全不是一回事。很多现场事故来自把“清零”误当“回零”。

### 报警复位 vs 重新使能

- 报警复位是消除驱动器故障状态
- 重新使能是恢复驱动输出

有些系统需要先复位报警，再清互锁，再重新使能，顺序不能乱。

### 到位 vs 稳定可加工

- 到位信号只是说明位置误差进入某个范围
- 稳定可加工还要看振动、机械稳定时间、真空/气缸状态、相机曝光条件

## 11. 一个足够实用的记忆方法

可以把所有术语按这 5 层去记：

1. 机械层：导程、背隙、行程、负载
2. 轴控层：回零、点动、定位、插补、加减速
3. 反馈层：编码器、光栅尺、跟随误差
4. 总线层：EtherCAT、PDO、SDO、CiA 402
5. 安全层：急停、限位、STO、联锁

只要你知道一个术语属于哪一层，理解速度会快很多。
