# 工控行业 C# 常用工具类

本次补充已在 `CodeDemo/CIM/src/Cim.DbAdapter/Utilities` 下新增一组可直接复用的工具类，适合 3C / 制造执行 / 设备联网 / PLC 对接场景：

## 已新增

- `ProtocolUtility.cs`
  - 大小端转换
  - 位读写
  - BCD 编码/解码
  - 简单帧组装
  - 字节字序翻转

- `ChecksumUtility.cs`
  - LRC
  - XOR
  - CRC16-Modbus
  - CRC16-CCITT
  - CRC32
  - MD5 / SHA1 / SHA256

- `EncodingUtility.cs`
  - Hex 与 byte[] 互转
  - ASCII 编解码
  - 二进制字符串显示
  - 设备编码标准化
  - 紧凑时间戳格式转换

- `StringUtility.cs`
  - Trim/判空/补位
  - Left/Right
  - 去空白
  - 仅保留字母数字
  - snake/pascal/camel 命名转换
  - 单行日志安全化
  - 忽略大小写比较
  - 分隔拆分
  - 设备编码格式校验
